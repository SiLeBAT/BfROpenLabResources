Learn
  • how to import a KNIME workflow into the KNIME desktop application
1
  • Download the workflow you would like to import, e.g. “FCL_Example.zip”
  • Right click on “LOCAL” in the “KNIME Explorer” view and select “Import KNIME Workflow”.
2
  • In the import dialog choose “Select file” and click the “Browse” button.
3
  • Select the downloaded zipped workflow and click “Open”.
4
  • In the Import dialog, you’ll see the selected workflow.
  • Click “Finish”.
5
  • In the KNIME workbench, expand “LOCAL” in the “KNIME Explorer” view and double click on the imported workflow.
  • The workflow should show up in the “Workflow Editor” now (see red square).
