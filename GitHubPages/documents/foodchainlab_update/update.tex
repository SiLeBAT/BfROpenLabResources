\section{2}
\begin{frame}
	\begin{center}
  		\includegraphics[height=0.6\textheight]{2.png}
	\end{center}
	\begin{itemize}
		\item You might get an error message saying "Cannot complete the install because one or more required items could not be found.".
		\item In that case it is likely, that your KNIME version is too old. So you must reinstall KNIME and FoodChain-Lab.
		\item If you do not get an error, proceed with step 3.
	\end{itemize}
\end{frame}
